积分表演绎

Posted by cainslayer on July 15, 2022 · 18 mins read

积分表的模板来自 网站 。本文对积分表的导出过程进行演绎。

\[\newcommand{\dif}{\mathrm d}\] \[\newcommand{\ddx}{\frac \dif {\dif x}}\]

Basic Forms

 

1.

\[\int {x^n \, \dif x} = \frac 1 {n+1} x^{n+1} , \, n \neq -1\]

  \(Proof:\)

\[\because \ddx x^n = \lim \limits_{h \to 0} \frac { (x+h)^n - x^n } h = \lim \limits_{h \to 0} \frac { (x^n + n \cdot h x^{n-1} + \omicron (h))) - x^n } h = n x^{n-1}\] \[\therefore \ddx \frac {x^{n+1}} {n+1} = x^n\]

  \(\square\)

2.

\[\int {\frac 1 x \, \dif x} = \ln | x |\]

  \(Proof:\)

\[\lim \limits_{v \to \pm \infty} \, (1 + \frac 1 v)^v = \mathrm e\] \[\Rightarrow \lim \limits_{u \to 0} \, (1+u)^{\frac 1 u} = \mathrm e\] \[\Rightarrow \lim \limits_{u \to 0} \frac {\ln (1+u)} u = 1\] \[\textup{if } x > 0\] \[\ddx \ln x = \lim \limits_{h \to 0} \frac {\ln (x+h) - \ln x} h = \lim \limits_{h \to 0} \frac {\ln ( 1 + \frac h x )} h = \frac 1 x\] \[\textup{if } x < 0\] \[\ddx \ln {(-x)} = - \lim \limits_{h \to 0} \frac {\ln ( 1 - \frac h x )} h = \frac 1 x\]

  \(\square\)

3.

\[\int u \, \dif v = uv - \int v \, \dif u\]

  \(Proof:\)

\[\because \ddx uv = \lim \limits_{h \to 0} \frac {u(x+h) v(x+h) - u(x) v(x)} h\] \[= \lim \limits_{h \to 0} \frac {u(x+h) (v(x+h) - v(x))} h + \lim \limits_{h \to 0} \frac {v(x) (u(x+h) - u(x))} h\] \[= u(x) \ddx v(x) + v(x) \ddx u(x)\] \[\therefore \dif uv = u \dif v + v \dif u\]

  \(\square\)

\[\int \frac 1 {ax+b} \dif x = \frac 1 a \ln | ax+b |\]

  \(Trivially\)

Integrals of Rational Functions

 

5.

\[\int \frac 1 {(x+a)^2} \, \dif x = - \frac 1 {x+a}\]

  \(Trivially\)

6.

\[\int (x+a)^n \dif x = \frac {(x+a)^{n+1}} {n+1}\]

  \(Trivially\)

7.

\[\int x (x+a)^n \dif x = \frac {(x+a)^{n+1} ((n+1)x - a) } {(n+1)(n+2)}\]

  \(Proof:\)

\[\int x (x+a)^n \dif x = \int (x+a)^{n+1} \dif x - a \int (x+a)^n \dif x\] \[= \frac {(x+a)^{n+2}} {n+2} - a \frac {(x+a)^{n+1}} {n+1}\]

  \(\square\)

8.

\[\int \frac 1 {1 + x^2} = \arctan x\]

  \(Proof:\)

\[\textup{let } \arctan x = u, \, u \in [ - \frac \pi 2, \frac \pi 2]\] \[\Rightarrow x = \tan u\] \[\Rightarrow 1 = \sec ^2 u \cdot \ddx \, u\] \[\Rightarrow \ddx \, u = \cos ^2 u = \frac 1 {1 + \tan ^ 2 u}\] \[\textup{i.e. } \ddx \, \arctan x = \frac 1 {1 + x^2}\]

  \(\square\)

9.

\[\int \frac 1 {a^2 + x^2} \dif x = \frac 1 a \arctan {\frac x a}\]

  \(Trivially\)

10.

\[\int \frac x {a^2 + x^2} \dif x = \frac 1 2 \ln (a^2 + x^2)\]

  \(Trivially\)

11.

\[\int \frac {x^2} {a^2 + x^2} \dif x = x - a \arctan {\frac x a}\]

  \(Proof:\)

\[\int \frac {x^2} {a^2 + x^2} \dif x = \int \dif x - a^2 \int \frac 1 {a^2 + x^2} \dif x\]

  \(\square\)

12.

\[\int \frac {x^3} {a^2 + x^2} \dif x = \frac 1 2 x^2 - \frac {a^2} 2 \ln (a^2 + x^2)\]

  \(Proof:\)

\[\int \frac {x^3} {a^2 + x^2} \dif x = \int (1 - \frac {a^2} {a^2 + x^2}) x \dif x = \int x \dif x - a^2 \int \frac x {a^2 + x^2} \dif x\]

  \(\square\)

13.

\[\int \frac 1 {a x^2 + b x + c} \dif x = \frac 2 {\sqrt {4 ac - b^2}} \arctan \frac {2 ax + b} {\sqrt {4 ac - b^2}}\]

  \(Proof:\)

\[a x^2 + b x + c = a ( (x + \frac b {2a})^2 + \frac {4 ac - b^2} {4 a^2} )\] \[\textup{In this case: } \Delta = b^2 - 4 ac < 0\] \[\textup{let } u = x + \frac b {2a}\] \[\int \frac 1 {a x^2 + b x + c} \dif x = \frac 1 a \int \frac 1 {u^2 + \frac {- \Delta} {4 a^2}} \dif u = \frac 2 {\sqrt {- \Delta}} \arctan \frac {2 au} {\sqrt {- \Delta}}\]

  \(\square\)

14.

\[\int \frac 1 {(x+a)(x+b)} \dif x = \frac 1 {b-a} \ln \frac {a+x} {b+x} , \, a \ne b\]

  \(Proof:\)

\[\textup{In this case: } a+x > 0 , \, b+x > 0\] \[\int \frac 1 {(x+a)(x+b)} \dif x = \frac 1 {b-a} (\int \frac 1 {x+a} \dif x - \int \frac 1 {x+b} \dif x )\]

  \(\square\)

15.

\[\int \frac x {(x+a)^2} \dif x = \frac a {a+x} + \ln | a+x |\]

  \(Proof:\)

\[\int \frac x {(x+a)^2} \dif x = \int \frac 1 {a+x} \dif x - a \int \frac 1 {(a+x)^2} \dif x\]

  \(\square\)

16.

\[\int \frac x {a x^2 + b x + c} \dif x = \frac 1 {2a} \ln | ax^2+bx+c | - \frac b {a \sqrt {4 ac - b^2}} \arctan \frac {2 ax + b} {\sqrt {4 ac - b^2}}\]

  \(Proof:\)

\[\int \frac x {a x^2 + b x + c} \dif x = \frac 1 {2a} \int \frac {2 ax + b} {ax^2 + bx + c} \dif x - \frac b {2a} \int \frac 1 {ax^2 + bx + c} \dif x\]

  \(\square\)

20.

\[\int x \sqrt{x-a} \, \dif x = \frac 2 5 (x-a)^{\frac 5 2} + \frac {2a} 3 (x-a) ^ {\frac 3 2}\]

  \(Proof:\)

\[\int x \sqrt{x-a} \, \dif x = \int (x-a)^{\frac 3 2} \, \dif x + a \int \sqrt {x-a} \, \dif x\]

  \(\square\)

21.

\[\int \sqrt {ax+b} \, \dif x = \frac 2 {3a} (ax+b)^{\frac 3 2} = \frac 2 3 (\frac b a + x) \sqrt {ax+b}\]

  \(Trivially\)

22.

\[\int (ax+b)^{\frac 3 2} \, \dif x = \frac 2 {5a} (ax+b)^{\frac 5 2}\]

  \(Trivially\)

23.

\[\int \frac x {\sqrt {x+a}} \dif x = \frac 2 3 (x-2a) \sqrt {x+a}\]

  \(Proof:\)

\[\int \frac x {\sqrt {x+a}} \dif x = \int \sqrt {x+a} \, \dif x - a \int \frac 1 {\sqrt {x+a}} \dif x\]

  \(\square\)

24.

\[\int \sqrt {\frac x {a-x}} \dif x = - \sqrt {x (a-x)} + a \arctan \sqrt {\frac x {a-x}}\]

  \(Proof:\)

\[\textup{As for intergral } \int \frac 1 {(1+t^2)^2} \dif t \textup{, let } t = \tan u\] \[\int \frac 1 {(1+t^2)^2} \dif t = \int \cos ^2 u \dif u = \frac u 2 + \frac {\sin 2u} 4 = \frac {\arctan t} 2 + \frac t {2(1+t^2)}\] \[\textup{let } t = \sqrt {\frac x {a-x}}\] \[\int \sqrt {\frac x {a-x}} \dif x\] \[= 2 a \int \frac {t^2} {(1+t^2)^2} \dif t\] \[= 2 a \int \frac 1 {1+t^2} \dif t - 2 a \int \frac 1 {(1+t^2)^2} \dif t\] \[= a \arctan t - \frac {at} {1+t^2}\] \[\textup{i.e. } \int \sqrt {\frac x {a-x}} \dif x = - \sqrt {x (a-x)} + a \arctan \sqrt {\frac x {a-x}}\]

  \(\square\)

25.

\[\int \sqrt {\frac x {a+x}} \dif x = \sqrt {x (a+x)} - a \ln (\sqrt x + \sqrt {x+a})\]

  \(Proof:\)

\[\textup{let } t = \sqrt {\frac x {a+x}} \textup{, then } x = \frac a 2 (\frac 1 {1-t} + \frac 1 {1+t}) - a\] \[\textup{In this case: } a > 0\] \[\int \sqrt {\frac x {a+x}} \dif x\] \[= \frac a 2 (\int \frac t {(t-1)^2} \dif t - \int \frac t {(t+1)^2} \dif t)\] \[= \frac a 2 (\int \frac 1 {(t-1)^2} \dif t + \int \frac 1 {(t+1)^2} \dif t - 2 \int \frac 1 {1-t^2} \dif t)\] \[= a \frac t {1-t^2} - a \, \mathrm{arctanh} \, t\] \[\textup{i.e. } \int \sqrt {\frac x {a+x}} \dif x = \sqrt {x (a+x)} - a \ln (\sqrt x + \sqrt {x+a}) + C\]

  \(\square\)

26.

\[\int x \sqrt{ax+b} \, \dif x = (\frac {-4b^2} {15a^2} + \frac {2bx} {15a} + \frac {2x^2} 5) \sqrt {ax+b}\]

  \(Proof:\)

\[\textup{let } t = ax+b\] \[\int x \sqrt{ax+b} \, \dif x = \frac 1 {a^2} \int t \sqrt t \, dt - \frac b {a^2} \int \sqrt t \dif t = (\frac 2 {5a^2} t^2 - \frac {2b} {3a^2} t) \sqrt t\] \[\textup{i.e. } \int x \sqrt{ax+b} \, \dif x = (\frac 2 {5a^2} (ax+b)^2 - \frac {2b} {3a^2} (ax+b)) \sqrt {ax+b}\]

  \(\square\)

29.

\[\int \sqrt {x^2 \pm a^2} \, \dif x = \frac 1 2 x \sqrt {x^2 \pm a^2} \pm \frac {a^2} 2 \ln |x + \sqrt {x^2 \pm a^2}|\]

  \(Proof:\)

\[\textup{In this case: } a > 0\]

  \(\textup{Case +: }\)

\[\int \cosh ^2 t \, \dif t = \int \frac {1 + \cosh 2t} 2 \dif t = \frac t 2 + \frac {\sinh 2t} 4\] \[\textup{let } x = a \sinh t\] \[\int \sqrt {x^2 + a^2} \, \dif x\] \[= a^2 \int \cosh ^2 t \, \dif t\] \[= \frac {a^2} 2 (t + \sinh t \cosh t)\] \[\textup{i.e. } \int \sqrt {x^2 + a^2} \, \dif x = \frac {a^2} 2 \, \mathrm{arcsinh} \, \frac x a + \frac 1 2 x \sqrt {x^2 + a^2}\]

  \(\textup{Case -: }\)

\[\int \sinh ^2 t \, \dif t = \int \frac {-1 + \cosh 2t} 2 \dif t = - \frac t 2 + \frac {\sinh 2t} 4\] \[\textup{(1) When } x > 0 \textup{, let } x = a \cosh t\] \[\int \sqrt {x^2 - a^2} \, \dif x\] \[= a^2 \int \sinh ^2 t \, \dif t\] \[= \frac {a^2} 2 (- t + \sinh t \cosh t)\] \[\textup{i.e. } \int \sqrt {x^2 + a^2} \, \dif x = - \frac {a^2} 2 \, \mathrm{arccosh} \, \frac x a + \frac 1 2 x \sqrt {x^2 - a^2}\] \[\textup{(2) When } x < 0 \textup{, let } x = - a \cosh t\] \[\int \sqrt {x^2 - a^2} \, \dif x\] \[= - a^2 \int \sinh ^2 t \, \dif t\] \[= \frac {a^2} 2 (t - \sinh t \cosh t)\] \[\textup{i.e. } \int \sqrt {x^2 + a^2} \, \dif x = \frac {a^2} 2 \, \mathrm{arccosh} \, (- \frac x a) + \frac 1 2 x \sqrt {x^2 - a^2}\]

  \(\square\)

30.

\[\int \sqrt {a^2 - x^2} \, \dif x = \frac 1 2 x \sqrt {a^2 - x^2} + \frac 1 2 a^2 \arctan \frac x {\sqrt {a^2 - x^2}}\]

  \(Proof:\)

\[\textup{In this case: } a > 0\] \[\textup{let } x = a \cos t, \, t \in [ - \frac \pi 2, \frac \pi 2]\] \[\int \sqrt {a^2 - x^2} \, \dif x\] \[= - a^2 \int \sin ^2 t \, \dif t\] \[= \frac {a^2} 2 (\sin t \cos t - t)\] \[\textup{i.e. } \int \sqrt {a^2 - x^2} \, \dif x\] \[= \frac 1 2 x \sqrt {a^2 - x^2} - \frac {a^2} 2 \arccos \frac x a\] \[= \frac 1 2 x \sqrt {a^2 - x^2} - \frac {a^2} 2 \arctan \frac {\sqrt {a^2 - x^2}} x\]

  \(\square\)

31.

\[\int x \sqrt {x^2 \pm a^2} \, \dif x = \frac 1 3 (x^2 \pm a^2)^{\frac 3 2}\]

  \(Trivially\)

32.

\[\int \frac 1 {\sqrt {x^2 \pm a^2}} \dif x = \ln |x + \sqrt {x^2 \pm a^2}|\]

  \(Proof:\)

\[\textup{In this case: } a > 0\]

  \(\textup{Case +: }\)

\[\textup{let } x = a \sinh t\] \[\int \frac 1 {\sqrt {x^2 + a^2}} \, \dif x = \int \, \dif t = t\] \[\textup{i.e. } \int \frac 1 {\sqrt {x^2 + a^2}} \, \dif x = \mathrm{arcsinh} \, \frac x a\]

  \(\textup{Case -: }\)

\[\textup{(1) When } x > 0 \textup{, let } x = a \cosh t\] \[\int \frac 1 {\sqrt {x^2 - a^2}} \, \dif x = \int \, \dif t = t\] \[\textup{i.e. } \int \frac 1 {\sqrt {x^2 - a^2}} \, \dif x = \mathrm{arccosh} \, \frac x a\] \[\textup{(2) When } x < 0 \textup{, let } x = - a \cosh t\] \[\int \frac 1 {\sqrt {x^2 - a^2}} \, \dif x = - \int \, \dif t = - t\] \[\textup{i.e. } \int \frac 1 {\sqrt {x^2 - a^2}} \, \dif x = - \mathrm{arccosh} \, (- \frac x a)\]

  \(\square\)

33.

\[\int \frac 1 {\sqrt {a^2 - x^2}} \dif x = \arcsin \frac x a\]

  \(Proof:\)

\[\textup{let } x = a \sin t, \, t \in [ - \frac \pi 2, \frac \pi 2]\] \[\int \frac 1 {\sqrt {a^2 - x^2}} \dif x = \int \, \dif t = t\]

  \(\square\)

\[\int \frac 1 {(a^2 + x^2)^{\frac 3 2}} \, \dif x = \frac x {a^2 \sqrt {a^2 + x^2}}\]

  \(Proof:\)

\[\textup{In this case: } a > 0\] \[\textup{let } x = a \sinh t\] \[\int \frac 1 {(a^2 + x^2)^{\frac 3 2}} \, \dif x = \frac 1 {a^2} \int \mathrm{sech} ^2 t \, \dif t = \frac 1 {a^2} \tanh t\] \[\textup{i.e. } \int \frac 1 {(a^2 + x^2)^{\frac 3 2}} \, \dif x = \frac 1 {a^2} \cdot \frac {x/a} {\sqrt {1 + (x/a)^2}}\]

  \(\square\)
